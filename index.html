Assignment 1: Create Your Favorite Recipe Page

Instructions

Follow these steps to create an HTML page about your favorite recipe. Use only the HTML elements we've learned in class.

What You Need to Include:

  1. Page Title: Give your HTML page a proper title using the <title> tag
  2. Main Heading: Use an <h1> tag for the name of your recipe
  3. Recipe Image: Add an image of your dish using the <img> tag (you can use any food image from the internet)
  4. Description: Write a short paragraph describing why this recipe is special to you
  5. Ingredients List: Create an unordered list (<ul>) of all ingredients needed
  6. Instructions: Create an ordered list (<ol>) with step-by-step cooking instructions
  7. Additional Section: Add a section with an <h2> heading called "Tips" with helpful cooking tips
  8. External Link: Include a link to a cooking website or food blog you like
  9. Contact Form: Add a simple form where visitors can send you feedback about the recipe

Form Requirements:

  • Name input field with proper label
  • Email input field with proper label
  • Comments text area with proper label
  • Submit button

HTML Elements You Must Use:

  • <h1>, <h2>, <h3> (headings)
  • <p> (paragraphs)
  • <img> (images with alt text)
  • <ul> and <ol> (lists)
  • <a> (links)
  • <form>, <input>, <label>, <textarea> (forms)
  • <br> (line breaks if needed)
  • <hr> (horizontal rule to separate sections)

Tips for Success:

  • Remember to include alt text for your images
  • Make sure all your form inputs have labels
  • Use proper nesting for your HTML elements
  • Test your links to make sure they work
  • Keep your content organized and easy to read

Example Structure:

Your page should have a structure similar to this:

  1. Page title (in head section)
  2. Recipe name (h1)
  3. Recipe image
  4. Description paragraph
  5. Ingredients heading (h2) and list
  6. Instructions heading (h2) and numbered steps
  7. Tips section (h2) with helpful tips
  8. External link to cooking resource
  9. Feedback form

Save your file as: my-recipe.html
